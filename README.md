# CaliTrack Mobile

React Native mobile app for structured workout tracking, built with 
TypeScript and Expo.

## Features
- Exercise logging and session management
- Training history and progress tracking
- Weekly analytics with time-series visualisation
- Client-side telemetry capturing p50/p95 API latency metrics
- Supabase Auth integration for user authentication

## Tech Stack
- React Native / Expo
- TypeScript
- NativeWind (Tailwind CSS)
- Supabase Auth

## Related
- [CaliTrack Backend](https://github.com/Vish2501/CaliTrack-backend) 
  — Spring Boot REST API

## Setup
```bash
npm install
npx expo start
```

> Note: Requires the CaliTrack backend to be running locally. 
> See backend repo for setup instructions.
